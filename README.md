# hypercode-java25

Repositório de desafios Java (do básico ao especialista).  
Cada participante cria uma pasta dentro de `users/` com o seu nome de usuário GitHub e registra o progresso nos arquivos `profile.md` e `progress.md`.

Estrutura:
- `challenges/` — desafios organizados por nível (basic, intermediate, advanced, expert)
- `users/` — pastas dos participantes
- `docs/` — site gerado com leaderboard (publicado via GitHub Pages)

Quer contribuir?
1. Fork o repositório.
2. Crie a pasta `users/<seu-username>/`.
3. Adicione `profile.md` e `progress.md` (use os templates em CONTRIBUTING.md).
4. Abra um Pull Request para `main`.

Leia `CONTRIBUTING.md` para detalhes.
