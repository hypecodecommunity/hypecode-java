# basic-001 — Introdução ao Java

Objetivo:
- Criar um programa `HelloWorld.java` que imprima `Hello, Java!`.

Requisitos:
- Arquivo fonte: `HelloWorld.java`
- Pacote: (opcional) não obrigatório
- Compilação: `javac HelloWorld.java`
- Execução: `java HelloWorld`

Entrega:
- Coloque a solução em `users/<seu-username>/solutions/basic-001/`
- Nome do arquivo: `HelloWorld.java`
